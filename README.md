## 🚀 Sobre o Projeto

O **Motta Music** é um player de música mobile moderno criado para fins de portfólio pessoal. O objetivo principal do projeto é aplicar os padrões mais recentes recomendados pela Google para o desenvolvimento Android nativo, garantindo uma interface fluida, transições reativas e uma excelente experiência de usuário ao navegar por faixas de áudio e playlists.

## ✨ Funcionalidades Principais

* **Navegação por Abas (Tab Row):** Separação intuitiva entre a lista completa de **Músicas** e as **Playlists** do usuário.
* **Barra de Pesquisa Dinâmica:** Campo de busca funcional no topo para filtrar faixas ou artistas em tempo real.
* **Scroll Alfabético Rápido (Fast Scroll):** Índice lateral de letras (A-Z) para ajudar o usuário a navegar rapidamente por listas extensas de músicas.
* **Gerenciamento de Playlists:** Atalho rápido para criação de novas coleções de faixas musicais diretamente pela interface do aplicativo.
* **Miniplayer Persistente:** Barra inferior de controle que exibe a faixa atual em reprodução, permitindo pausar/dar play e expandir os controles sem interromper a navegação no app.
* **Identidade Visual Dark Mode:** Tema escuro nativo elegante com detalhes em tons roxos e cinzas, otimizando o conforto visual.

## 🛠️ Tecnologias e Arquitetura

O projeto foi construído utilizando as ferramentas de ponta do desenvolvimento Android:

* **Kotlin (100%):** Linguagem de programação principal adotada pela Google para apps robustos.
* **Jetpack Compose:** Kit de ferramentas moderno para construção de interfaces declarativas, dinâmicas e reativas sem o uso de arquivos XML tradicionais.
* **Material Design 3:** Implementação dos padrões visuais mais recentes da Google para componentes de interface e tipografia.
* **Gradle Kotlin DSL:** Configuração e gerenciamento de dependências limpo e moderno (`build.gradle.kts`).
